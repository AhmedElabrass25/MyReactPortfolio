## üöÄ Getting Started

### Prerequisites
- [Node.js](https://nodejs.org/) (v16 or higher)
- [npm](https://www.npmjs.com/) or [yarn](https://yarnpkg.com/)

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/AhmedElabrass25/react-portofolio.git
   ```
2. Navigate to the project directory:
   ```bash
   cd react-portofolio
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the development server:
   ```bash
   npm run dev
   ```

### Environment Setup
To enable the Contact Form, create a `.env` file or update the credentials in `src/components/Contact/Contact.jsx` with your EmailJS:
- `SERVICE_ID`
- `TEMPLATE_ID`
- `PUBLIC_KEY`
